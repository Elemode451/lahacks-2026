# Seratone Design System

## Mission
Music discovery through predicted brain-response similarity. Recommending music based on vibes and cortical overlap rather than algorithmic trends or metadata.

## Aesthetic
**Architectural Minimalism meets Glassmorphism.** Stark, whitespace-heavy layouts driven by bold typography, contrasted with soft, blurred 3D elements and semi-transparent overlays. Straightforward, blunt, but inviting.

## Core Principles

1. **Whitespace is structure** — The off-white background gives the 3D brain mesh space to breathe. Never fill the page.
2. **Color is data** — Colors are reserved strictly for data visualization, highlights, and active states.
3. **Motion is meaning** — Every animation communicates a state change. Use Apple-style easing for all transitions.
4. **Glass over chrome** — UI elements use glassmorphism (frosted glass) with `backdrop-filter: blur()`.
